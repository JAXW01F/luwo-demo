# LUWO Golden Spiral Wave Generator

**LUWO = Layered Unified Waveforms & Oscillations**

This repository powers a **private LUWO generator server** and a small public demo of the LUWO image generator.

The engine transforms my custom **LUWO Golden Spiral** geometry into layered unified waveforms and oscillation patterns, exported as PNG images for print, design, or further processing.  
The underlying math and full engine remain **proprietary**.

## Concept: LUWO (Layered Unified Waveforms & Oscillations)

LUWO is a geometry system for generating:

- **Layered** — multiple nested contours derived from a single spiral logic  
- **Unified** — all waves share a common underlying curvature system  
- **Waveforms** — smooth organic paths suitable for art, UI, or signal metaphors  
- **Oscillations** — shapes that feel like frozen motion, vibration, or flow  

The **LUWO Golden Spiral** is the core curve behind these forms.  
Standard spirals don’t create these exact behaviors or silhouettes.

Formal math notes, PDFs, and extended documentation live in a **separate private codebase**.
